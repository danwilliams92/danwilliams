Landing Page

It has several sections, and a call-to-action that brings up a modal contact form (which in this case is front-end-only, so don't be disappointed if you fill in the form and nothing happens!).

Restaurant Website

In this project I designed a mobile-first website for a fictional French restaurant, using CSS3 and some JavaScript for functionality. Minus the contact page, I tried to make this work as a real small-scale restaurant website would. For the layout I used Flexbox: at the time I read mixed things about browser support for CSS Grid, so I played it safe and used Flex.

For the future I need to make the pages responsive for tablet devices, as well as clean up the JavaScript code because in retrospect it is a little sloppy

Current Weather App

In this project I use jQuery and AJAX to access the Openweathermap API. The user can get weather data either for a particular city, or for their current location (which is found through Geocode).

This is my first project that uses API calls, and I have much more to learn, however this was still a rewarding project!
